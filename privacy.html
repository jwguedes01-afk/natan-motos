Política de Privacidade

Última atualização: 6 de junho de 2026

1. Informações que Coletamos

A Natan Motos coleta informações através de:

  • Formulário de Contato: Nome, telefone e mensagem quando você entra em contato conosco via WhatsApp
  • Google Analytics: Dados de navegação, incluindo páginas visitadas, tempo gasto no site, navegador e localização aproximada (cidade/país)
  • Cookies: Identificadores técnicos para melhoria da experiência do usuário

2. Como Usamos Suas Informações

Utilizamos os dados coletados para:

  • Responder suas solicitações de contato via WhatsApp
  • Melhorar o site através de análises de comportamento do usuário
  • Entender quais páginas e serviços são mais interessantes para os visitantes
  • Manter a segurança e funcionamento do site

3. Google Analytics

Utilizamos Google Analytics para coletar dados anônimos sobre o comportamento dos visitantes. Esses dados incluem:

  • Número de visitantes
  • Páginas mais visitadas
  • Tempo médio gasto no site
  • Tipo de dispositivo (mobile, desktop)
  • País/cidade aproximada

Você pode desabilitar o Google Analytics instalando a extensão de navegador "Google Analytics Opt-out" disponível em: https://tools.google.com/dlpage/gaoptout

4. Compartilhamento de Dados

Nós não compartilhamos seus dados pessoais com terceiros, exceto:

  • Google Analytics (anônimizado e agregado)
  • Google Tag Manager (para rastreamento técnico)
  • WhatsApp (quando você nos envia uma mensagem)

5. Seus Direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018), você tem direito a:

  • Acesso: Saber quais dados temos sobre você
  • Retificação: Corrigir dados incorretos
  • Exclusão: Solicitar a exclusão dos seus dados
  • Portabilidade: Receber seus dados em formato estruturado
  • Revogação de Consentimento: Deixar de consentir com a coleta de dados

6. Segurança

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados, incluindo:

  • Uso de HTTPS para todas as conexões
  • Headers de segurança HTTP
  • Ausência de transmissão de dados sensíveis não criptografados
